# react-native-secp256k1-urnm
React Native Secp255k1 lib
## Installation

```sh
npm install react-native-secp256k1-urnm
```

## Usage

```js
import { multiply } from "react-native-secp256k1-urnm";

// ...

const result = await multiply(3, 7);
```
